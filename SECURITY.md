# Security Policy

## Supported Versions

opskit is in early development (pre-1.0). Security fixes are applied to the latest released
version only.

| Version  | Supported |
| -------- | --------- |
| latest   | ✅        |
| < latest | ❌        |

## Reporting a Vulnerability

Please report security vulnerabilities **privately** — do **not** open a public issue.

Use GitHub's private vulnerability reporting for this repository:

- **https://github.com/Prabhakar-cg/opskit/security/advisories/new**

We aim to acknowledge new reports within **5 business days**. After triage and a fix, we will
coordinate disclosure and credit reporters who wish to be named.

## Scope

opskit is a **read-only diagnostic** tool and intentionally ships no offensive capabilities
(see the project constitution, Article X). Reports about the tool being *used* to attack third
parties are out of scope; reports about vulnerabilities *in* opskit or its dependencies are in
scope.
